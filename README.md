# Opencode-Local
Hardware objetivo: Mac Studio M4, 64 GB Unified Memory Modelos: Qwen3.6-35B-A3B · Qwen3.5-9B · Gemma-4-26B-A4B Objetivo final: Un único endpoint OpenAI-compatible en el puerto 8080, gestionado por llama-swap, que carga/descarga modelos bajo demanda. Listo para usarse con oh-my-openagent u otras herramientas.
